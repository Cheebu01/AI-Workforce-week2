# Creating a User

Sign in to Microsoft 365 Admin Center.
Go to Users > Active users > Add a user. Fill in user details and click Next.

# Assigning a License 
In the user setup, choose a license (e.g., Microsoft 365 E3). Licenses give access to apps like Word, Teams, OneDrive, etc.

# Benefits of Assigning a License
Productivity tools: Access to Word, Excel, PowerPoint, Outlook. Collaboration: Teams, SharePoint, OneDrive for file sharing. Security: Advanced security features like MFA, data protection. Compliance: Tools for legal, HR, and data management. Mobility: Work from anywhere on any device.
